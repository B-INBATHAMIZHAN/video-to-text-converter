# AI Video-to-Text Converter

An offline AI-powered video-to-text converter built using Python and OpenAI Whisper.

## Features
- Convert video speech to text
- Offline processing
- English speech transcription
- No paid subscription needed
- No sign-up required

## Technologies Used
- Python
- Whisper AI
- FFmpeg

## Why I Built This
I faced a real-world problem where many video-to-text websites required payment, sign-up, or limited credits. Instead of depending on them, I built my own offline solution using AI.

## How to Run

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Run the script:

```bash
python v.py
```
